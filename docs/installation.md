# Installation

## Stable release

To install Instructor Examples, run this command in your terminal:

```sh
uv add instructor_examples
```

Or if you prefer to use `pip`:

```sh
pip install instructor_examples
```

## From source

The source files for Instructor Examples can be downloaded from the [Github repo](https://github.com/dgmouris/instructor_examples).

You can either clone the public repository:

```sh
git clone https://github.com/dgmouris/instructor_examples
```

Or download the [tarball](https://github.com/dgmouris/instructor_examples/tarball/main):

```sh
curl -OJL https://github.com/dgmouris/instructor_examples/tarball/main
```

Once you have a copy of the source, you can install it with:

```sh
cd instructor_examples
uv sync
```
